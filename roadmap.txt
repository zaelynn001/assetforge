# Rev 1.0.0

AssetForge Development Roadmap (Rev 0.0.1)

M1 — Bootstrap & Plumbing (Rev 0.0.1)
------------------------------------
- Set up repo structure, migrations, and logging.
- Implement SQLite wrapper with WAL mode and migration runner.
- Create minimal PySide6 MainWindow (maximized) with placeholder panels.
- Seed default hardware_types.
- Smoke-test DB creation and asset_tag generation.

M2 — Core Domain & Repositories (Rev 0.2.0)
-------------------------------------------
- Implement repositories for hardware_items, types, locations, users, groups, attributes, and updates.
- Add CRUD operations with validation and updated_at_utc management.
- Add attribute key/value helpers.
- Implement audit recording for changes.

M3 — UI Shell & Grid (Rev 0.4.0)
---------------------------------
- Build main layout: filters (left), grid (center), tabs (right).
- Display key columns: asset_tag, name, model, type, MAC, location, user/group, updated.
- Add selection handling and detail display.
- Implement New Item dialog and Edit modal.
- Add search bar and scanner-friendly focus handling.

M4 — Assignment, Movement, Attributes (Rev 0.6.0)
-------------------------------------------------
- Add Assign User/Group and Move Location dialogs.
- Build Attributes tab with editable key/value pairs.
- Implement attribute validation and duplicate prevention.
- Enable bulk attribute import from CSV.

M5 — History & Audit UX (Rev 0.7.0)
-----------------------------------
- Display history of each item from item_updates.
- Implement before/after diff highlighting.
- Allow manual “Audit note” entries.

M6 — Barcode/Asset-Tag Flow & Search (Rev 0.8.0)
------------------------------------------------
- Implement USB barcode stream detection.
- Jump-to-item by asset_tag or MAC.
- Add advanced search syntax (mac:, type:, loc:).

M7 — Export & Backup (Rev 0.9.0)
--------------------------------
- Add .xlsx export (openpyxl): grid and attributes sheets.
- Implement Backup script using git commit/push.
- Integrate “Backup Now” action into menu.

M8 — Packaging & Install (Rev 1.0.0)
------------------------------------
- Build PyInstaller one-folder bundle.
- Create .desktop launcher and install script.
- Add first-run setup and config validation.

M9 — QA, Polish & Docs (Rev 1.1.0)
----------------------------------
- Test uniqueness constraints, tagging logic, and performance.
- Improve error handling and logging output.
- Add user documentation and screenshots.
